Le but de l'atelier sera de comprendre les mecanismes de bases de l'arduino en plusieurs etapes.


1 Programmer l'arduino indėpendamment :
Faire clignoter une led.

2 Programmer l'arduino avec un input simple :
Faire clignoter une led selon un potard (potentiometre).

3 Programmer l'arduino avec un input evoluė (pc).
Faire clignoter une led a partir de son pc.

4 Rėcupėrer un input de l'arduino sur le pc :
Faire bouger une animation sur le pc en fonction d'un potard (potentiometre) reliė a l'arduino.
